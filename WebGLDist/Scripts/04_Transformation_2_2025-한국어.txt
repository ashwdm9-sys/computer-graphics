3D에서 로테이션을 계산하는데 효율적이고 안정적인 방법 중에 하나는
Quaternion을 이용하는 것입니다
Quaternion은4차원의 vector (q0, q1, q2, q3) 로써
하나의 scalar q0 와 하나의 3D vector q = (q1, q2, q3) 로 
이루어져 있다고 볼 수 있습니다
Quaternion의 이론은 상당히 복잡한 것들이 많이 있습니다만,
우리 코스에서는 rotation 의 계산에 사용할 최소한의 것들만
소개하도록 하겠습니다
먼저 두 개의 quaternion,  a = (q0, q) 와  b = (p0, p) 가 주어졌습니다
두 quaternion 의 합 a + b 는 정상적인 4D vector 들의 합인데
대응하는 element 들을 더하여 개선합니다
두 quaternion 의 곱 ab는 여기에 주어진 식과 같이 계산됩니다
여기에는 a와 b가 포함하는 3D 벡터들
q와 p의 dot product 와 cross product 가 사용되었습니다
quaternion 의 크기 magnitude, size 는 정상적인 4D 벡터의 크기입니다
quaternion 의 곱셈에 대한 inverse, 즉,  a^{-1} 는 이 식과 같이 정의됩니다
a 와 a^{-1} 의 곱은 곱셈에 대한 항등원 (1, 0) = (1, (0, 0, 0)) 이 됩니다
quaternion 을 사용하여 3D point 를 임의의 축에 대해 rotate 할 수 있습니다
먼저 rotation 을 할 3D 포인트 u = (x, y, z) 로
quaternion p = (0, u) 를 만듭니다
rotation axis 가 unit vector v 로 주어집니다
이제 v 축에 관해 u를 theta 만큼 rotate 한 결과인 u' 을 구하는 방법,
즉 u' = R_v (theta) u 를 보겠습니다
먼저 unit quaternion r 을 만듭니다
그리고 r^{-1} 을 다음과 같이 정의합니다
r^{-1} 도 unit quaternion 입니다
그러면 p' = (0, u') 은 다음 식에 의해 구해집니다:  p' = r p r^{-1}
quaternion과 rotation matrix form 간에는 서로 컨버전이 가능합니다
